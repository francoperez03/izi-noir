# Solana Anchor Development Skill

## Overview

This skill provides expertise in Anchor-based Solana program development for the IZI-NOIR project.

## Capabilities

- Program architecture and design patterns
- Account data structures and validation
- Instruction handling and error management
- Cross-program invocations (CPI)
- Testing strategies with Anchor

## When to Use

Use this skill when:
- Designing new Solana programs
- Implementing account structures
- Writing program instructions
- Setting up tests for Solana programs
- Debugging Anchor-related issues

## Key Principles

1. **Security First**: Always validate all accounts and inputs
2. **Minimal Trust**: Verify ownership and signer constraints
3. **Gas Efficiency**: Optimize for compute units
4. **Testability**: Design for comprehensive testing
